\begin{ex}[]
    Sia dato l'oscillatore smorzato 
    \[
	\frac{\text{d} ^2x}{\text{d} t^2} + \alpha\frac{\text{d} x}{\text{d} t} + x = 0 \quad  (\alpha >0) 
    .\] 
    Dimostrare che l'unico stato non-Wandering è $\v{V}_s = \begin{pmatrix} 0 \\ 0\end{pmatrix}$.
\end{ex}
